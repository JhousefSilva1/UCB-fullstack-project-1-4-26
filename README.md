## 🧱 1. HTML – Estructura de la Web

HTML define la estructura de una página.

### Ejemplo básico:

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Primera Página</title>
</head>
<body>
    <h1>Hola Mundo 🌎</h1>
    <p>Mi primera página web</p>
</body>
</html>
